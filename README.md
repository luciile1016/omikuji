# 浅草寺 御神籤 おみくじ

一個以日本浅草寺觀音籤為靈感製作的線上求籤網站。

🔗 **[點此求籤](https://luciile1016.github.io/omikuji/)**

---

## 關於這個專案

某次在浅草寺抽到了一支吉籤，於是做了這個網站。

網站收錄了浅草寺全部 100 支觀音籤，每支籤均包含籤詩原文、文言解讀、籤詩逐句說明，以及願望、疾病、待人、旅行等各項運勢分析。若抽到凶籤，還可以體驗將籤結於みくじ掛け的互動動畫——就像在寺廟裡把煩惱交付給觀音菩薩一樣。

## 求籤須知

- **一事一籤** — 一次只專注問一件事，多個願望請分開求取
- **清晰提問** — 在心中默念您的姓名、生日與具體困惑
- **落子无悔** — 無論大吉或凶籤，都是當下的溫柔預警

## 籤的種類

浅草寺觀音籤共有七種吉凶等級：

`大吉` `吉` `半吉` `小吉` `末吉` `末小吉` `凶`

## 技術說明

- 純靜態網頁，HTML / CSS / JavaScript
- 無後端、無框架、無資料庫
- 部署於 GitHub Pages

## 圖片來源

籤文圖片為浅草寺觀音籤實物掃描，僅供個人學習與文化分享用途。

# Senso-ji Temple Omikuji

An interactive fortune-telling website inspired by the traditional Omikuji at Senso-ji Temple in Asakusa, Tokyo.

🔗 **[Draw your fortune](https://luciile1016.github.io/omikuji/)**

---

## About

This project was born from a personal visit to Senso-ji Temple, where I drew a Kyo (bad luck) fortune slip.

The site includes all 100 original Senso-ji fortune slips, each with the original verse in classical Japanese, a detailed interpretation in Traditional Chinese, line-by-line explanations, and fortune readings across categories like wishes, health, love, and travel. If you draw a Kyo slip, you can also experience an interactive animation of tying your slip to the mikuji-kake rack — just like leaving your worries in the hands of Kannon at the temple.

## Before You Draw

- **One question at a time** — Focus on a single concern per draw
- **Be specific** — Silently recite your name, birthday, and your question
- **Accept the result** — Every slip, good or bad, is a gentle message for the present moment

## Fortune Types

Senso-ji Omikuji has seven levels of fortune:

`Daikichi (Great Luck)` `Kichi (Luck)` `Hankichi (Half Luck)` `Shokichi (Small Luck)` `Suekichi (Future Luck)` `Sueshokichi (Future Small Luck)` `Kyo (Bad Luck)`

## Tech

- Plain HTML / CSS / JavaScript — no frameworks, no backend
- Hosted on GitHub Pages

Fortune slip images are scanned from original Senso-ji Omikuji for personal and cultural appreciation only.
